Facility Resources
=================

This is the FACILITY RESOURCES page.


  • Scheduling Calendar
    • Master calendar
    • Shows all volunteer calendars
    • Shows all homeless calendars
    • Searchable
    • Export filtered (customized) calendars for printing or as PDF.
  • Contact List
    • Contact information for all participants
  • Application Manager
    • Pre-sorted, filtered applications for Volunteers
      • Requirements for qualification can be customized by administrator
      • Applications can be sorted by location, skills, interests, experience
    • Pre-sorted, filtered applications for Assistance requests
      • Applications can be prioritized by needs
      • Followup communication can be initiated from within a profile.
    • Profile Pages
      • Administrators can process applications and create profiles
      • Logs
        • Information on application
        • Communication with administration
        • Open / Fulfilled requests
        • Internal Notes
      • Schedule Events
        • Automatically adds events to the appropriate calendar(s)
        • Triggers alert messages to appropriate users.
  • Online Community
    • Live Chat
    • Forums
      • Volunteers shares storied of challenges and resolutions to help other volunteers in similar situations.
      • Private administration level rooms for working with other organizations facilitating the program.
